# Phones_Analysis
Data analytics project to understand phone market using microsoft excel, python, Power BI.

**Overview**
This project demonstrates a complete data analytics workflow — from loading and exploring a dataset in Python to building a Power BI dashboard. It highlights essential skills required for a Analyst role, including data cleaning, EDA, visualization.

**Dataset**
Filename: phones_data 
Format: CSV
Rows: 1500
Columns: 22

**Tools & Technologies**
Python (Pandas, NumPy) – Data loading, EDA, cleaning
Power BI – Dashboard creation
GitHub – Version control & documentation

**Steps Followed**
1️⃣ Load Dataset in Python: Imported dataset using Pandas Checked data types, summary statistics

2️⃣ Exploratory Data Analysis (EDA): Identified trends, patterns, and distributions Detected missing values, inconsistencies

3️⃣ Data Cleaning: Handled missing or incorrect values Created new calculated columns (if needed)

4️⃣ Power BI Dashboard: Loaded cleaned dataset into Power BI Built interactive visuals Created a professional dashboard showcasing insights
